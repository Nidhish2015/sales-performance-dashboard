# 🏏 IPL Analysis Dashboard (2008–2022)

An interactive Power BI dashboard that analyzes Indian Premier League (IPL) data from 2008 to 2022. The dashboard provides insights into team performance, player statistics, batting and bowling records, match outcomes, and season-wise trends through interactive visualizations.

---

# 📌 Project Overview

The objective of this project is to transform IPL historical data into meaningful insights using Power BI, SQL, and Excel.

The dashboard enables users to:

* Analyze team performance across seasons
* Compare player statistics
* Track batting and bowling records
* Explore season-wise trends
* Identify top performers
* Gain insights through interactive filtering and visualizations

---

# 🛠 Tech Stack

* Power BI
* SQL
* Excel
* DAX

* Economy Rate
* Best Bowling Figures

## Match Insights

* Venue-wise Results
* Match Outcome Distribution
* Season-wise Trends

* Player Filter
* Season Filter
* Dynamic Visualizations
* Cross-filtering
